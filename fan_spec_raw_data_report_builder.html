FAN SPECIFICATION → RAW DATA / REPORT 생성기

업로드한 PDF에서 FAN SPECIFICATION 페이지를 찾아 필요한 항목만 추출하고, SYS NO. 기준으로 RAW DATA를 만든 뒤 업로드된 REPORT.xlsx 템플릿 형식의 결과까지 생성합니다.
현재 버전은 사용자가 올린 샘플과 같은 동일 양식의 FAN SPEC 표에 맞춰져 있으며, 기본은 PDF 텍스트 추출, 필요하면 OCR로 보강할 수 있습니다. PDF에 보이는 순서(위→아래, 셀 내부는 상단→하단)를 우선 기준으로 읽고, 동일 SPEC은 하나의 탭으로 묶어서 E20:I20에 개별 SYS NO.를 넣고 Q'TY는 그 개수만큼 합산합니다.

선택된 PDF 없음
브라우저에서 PDF가 바로 열리는 일을 막기 위해, 이제 이 영역에 드래그 앤 드롭해도 업로드로 처리되도록 바꿨습니다.
예: S201/202 → S201, S202 / S208~210 → S208, S209, S210
원본 SYS NO., 원본 Q'TY, 페이지 번호를 추가로 표시합니다.
기본으로 현재 업로드한 REPORT.xlsx가 내장되어 있습니다. 필요하면 새 템플릿을 한 번 저장해 두고 계속 재사용할 수 있습니다.
기본 내장 템플릿 사용 중
현재 추출 항목:
YARD / HULL NO. / TYPE / SYS NO. / PS / Q'TY / USE / MAKER / TYPE(MOTOR) / OUTPUT / ELEC. SOURCE / IP / AIR FLOW RATE / RATING CURRENT / RPM
OCR 모드는 느릴 수 있습니다. 지금 올린 샘플처럼 텍스트가 살아있는 PDF는 Smart 또는 PDF 텍스트만 사용이 더 정확합니다.
